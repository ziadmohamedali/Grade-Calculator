# Grade Calculator

A simple, clean web app that checks your score and instantly shows your grade with color-coded results.

## ✨ Features
* **Modern UI:** Features a smooth floating input label and an animated 3D button.
* **Smart Validation:** Blocks empty inputs and invalid scores outside the 0–100 range.
* **Color Feedback:** Green for passing grades, orange for a bare pass, and red for failure.

## 🛠️ Built With
* HTML5
* CSS3
* JavaScript

## 🖥️ How to Run
Just open `index.html` in any web browser.
